selenium+python3环境搭建

  • python
  • selenium

selenium 是一个web的自动化测试工具，但是这次使用它是为写爬虫，这样在处理js上面可以省去很多的事，这次只讲环境的搭建，具体的使用我建议去看下虫师的博客

准备

准备工具如下：

安装python：具体的安装我就不多说了，如果你要用的包不是必须要依赖于python2的话，建议直接使用python3.5。

安装selenium：
无论是ubuntu还是windows，都可以使用pip或者easyz_install直接安装pip install selenium,如果在linux下面提示权限不够的话请使用root权限来安装

前提准备已经做完了，现在要准备相应的webdriver，这个决定于你是用的是什么浏览器了

Firefox：
新版已经把火狐的webdrver剥离了出来，所以需要单独下载。下载地址https://github.com/mozilla/geckodriver/releases

Google Chrome :
chrome浏览器本身就需要单独下载，下载地址https://chromedriver.storage.googleapis.com/index.html?path=2.27/,选择相应的平台下载自己的所需

之后把下载下来的文件放在一个文件夹中，之后把这个文件夹加入环境变量，或者直接放在已经加入了环境变量的文件夹中。

测试

完成了上面的几步，我们来测试下是不是已经可以正常使用了

浏览器的选择

Chrome、Firefox、IE、phantomjs
这里我重点介绍phantomjs，phantomjs是一个基于webkit的没有界面的浏览器，所以运行起来比完整的浏览器要高效。下载地址https://bitbucket.org/ariya/phantomjs/downloads/只需要选择相应平台下载就可以了
